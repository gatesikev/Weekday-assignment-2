The Campus Information Website is a simple and user-friendly web project designed to provide essential information about a campus. 
The website helps students, staff, and visitors easily access details about campus services, location, announcements, and general campus life.

This project was developed as part of a web development assignment to demonstrate the use of HTML and CSS for creating structured and well-organized websites.
The following are the features:

✅ Navigation bar with links to different sections

✅ Campus services section displaying available facilities

✅ Location section with campus address and details

✅ Gallery section showcasing campus images

✅ Announcements section for important notices

✅ Clean and simple layout for easy navigation
